# InternOSE

## Vue d'ensemble du projet
InternOSE est une plateforme de gestion de stages qui connecte les employeurs et les étudiants.  
Le projet comprend un frontend en **React + TypeScript + Tailwind CSS** et un backend en **Spring Boot + JPA + PostgreSQL + JWT**.

### Présentation du projet
**Objectif :** Créer une plateforme de gestion de stages qui connecte les employeurs et les étudiants.  

**Technologies :**
- Frontend ⇒ React + TypeScript + Tailwind CSS
- Backend ⇒ Spring Boot + JPA + PostgreSQL + JWT

**Architecture :** API REST + Interface utilisateur moderne

### Fonctionnalités principales
- Inscription/Connexion pour les employeurs et les étudiants  
- Isolation des données par utilisateur  
- Interface responsive et moderne  
- Tableau de bord employeur : création d’offres de stage  
- Tableau de bord étudiant : téléversement de CV
