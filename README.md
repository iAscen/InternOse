# 🎯 InternOSE

## 📋 **Vue d'ensemble du projet**

**InternOSE** est une plateforme de gestion de stages qui connecte les employeurs et les étudiants. Le projet comprend un frontend React avec TypeScript et un backend Spring Boot.

---

## 🚀 **Présentation du projet**

- **Objectif :** Une plateforme de gestion de stages pour connecter les employeurs et les étudiants
- **Technologies :**
    - Frontend ⇒ React + TypeScript + Tailwind CSS
    - Backend ⇒ Spring Boot + JPA + PostgreSQL + JWT
- **Architecture :** API REST + Interface utilisateur moderne

#### **Fonctionnalités principales**

- ✅ Inscription/Connexion pour les employeurs et les étudiants
- ✅ Isolation des données par utilisateur
- ✅ Interface responsive et moderne
- ✅ Tableau de bord pour les employeurs qui permet de :
    - Créer les offres de stages
- ✅ Tableau de bord pour les étudiants qui permet de :
    - Téléverser leur CV
